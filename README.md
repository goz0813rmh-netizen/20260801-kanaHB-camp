## 雨天・増水時の基本ルール
- 川が増水・濁流なら入らない
- 当日朝に現地へ川の状況を確認
- 川遊び不可なら温泉を早めるか周辺の屋内施設へ変更
- 大雨予報・警報級ならキャンセル規定を確認し旅行自体を再判断
