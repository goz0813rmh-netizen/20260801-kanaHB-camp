## 詳細タイムライン
### 8月1日（土）｜川遊び・BBQ・温泉
**08:00　自宅出発**
- 朝食は済ませて出発
- 水着は服の下に着ておく
- BBQ食材は基本的に前日までに準備

**09:00〜09:20　最終買い出し**
- 氷
- 飲み物
- 忘れた食材
- 子どもの昼食

**10:30〜11:00　やすらぎ村到着**
- チェックイン前なので臨時駐車場へ
- 受付方法と川の当日状況を確認

**11:00〜12:20　川遊び①**
- 初日はここがメイン
- 6歳・5歳ともライフジャケット着用
- 大人は必ず子どもの近くに付く

**12:20〜13:00　軽い昼食**
- おにぎり、パン、冷たい麺など
- BBQ前なので食べすぎない

**13:00〜13:25　着替え・受付準備**

**13:30　チェックイン**
- マッシュルームキャビンへ荷物移動
- 寝床と夜の照明を先に確認

**14:00〜15:30　川遊び②**
- 子どもの体力次第で短縮可
- 疲れていればキャビン周辺で休憩

**15:30〜16:30　着替え・休憩・BBQ準備**
- 子どもはおやつ
- 大人は火起こし・食材準備

**16:30〜18:15　BBQ**
- 暗くなる前に食事を終える
- 子ども用は最初に焼く

**18:15〜18:40　片付け**
- 生ゴミ・火元を処理
- 温泉セットを持って出発

**19:00〜20:00　日帰り温泉（候補確定中）**
- ここは営業時間・距離・料金を最終確認して確定する

**20:30　キャビン帰着**
- 水分補給
- 翌朝の服を準備

**21:30　子ども就寝目標**

### 8月2日（日）｜朝の自然遊び・ゆっくり撤収
**07:00　起床**

**07:30〜08:15　朝食**
- パン、ヨーグルト、バナナなど簡単に

**08:30〜10:00　朝の川辺遊び・散歩**
- 泳ぐ場合は当日の水位と天候を再確認

**10:00〜11:00　片付け開始**

**11:00〜12:00　予備時間**
- 子どもが遊びたければ最後の自然遊び
- 雨なら早めに撤収

**12:00〜12:40　最終片付け・積み込み**

**13:00　チェックアウト**

**13:30　昼食**

**15:00　帰路へ**

**17:00前後　帰宅目標**
