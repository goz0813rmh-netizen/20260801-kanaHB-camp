## 雨天・増水時プラン
### 小雨・川遊び不可
- 9:00出発は維持
- 現地で昼食、13:30チェックイン
- キャビンでカードゲーム・おやつ
- BBQ可能な天候なら早めに実施
- 温泉営業が確認できれば温泉を長めに楽しむ

### 強い雨・警報級
- 川には入らない
- 山間部への移動自体を再判断
- キャンセル規定を確認
- 「せっかく予約したから行く」はしない

### 屋内代替施設について
近隣の検索では、5歳・6歳が2〜3時間しっかり遊べる確実な大型屋内施設をまだ特定できていない。無理に名前だけ入れず、旅行直前の天気予報に合わせて、往路・帰路上の屋内施設を再検索する。
