## 車移動の考え方
### 往路
大饗周辺から奈良県南部へ向かう。実際のルートはGoogleマップが当日の渋滞・通行止めを反映して提示するルートを採用する。

### 出発時刻
**08:00出発を基本**とする。夏休み土曜のため、Googleマップの到着予測に30分程度の余裕を持つ。

### 買い出し方針
BBQの肉・野菜・朝食は前日までに自宅近くで購入。出発後の買い物は氷・飲み物・忘れ物だけにする。山間部に入ってから大型スーパーを探す計画にはしない。

## 温泉計画
### 第一候補：下市温泉秋津荘 明水館
やすらぎ村と同じ下市町内の温泉施設を第一候補にする。

**現時点の扱い：候補確定、8月1日夜の営業時間・日帰り入浴料金は最終確認待ち。**

Googleマップで施設間ルートをすぐ開けるようにした。BBQ後に移動するため、営業時間が合わない場合は無理に温泉を入れず、施設シャワーへ切り替える。

### 温泉の判断期限
**旅行1週間前までに8月1日の営業を再確認。**

## 雨天・増水時の基本ルール
- 川が増水・濁流なら入らない
- 当日朝に現地へ川の状況を確認
- 川遊び不可なら、温泉を早めるか周辺の屋内施設へ変更
- 大雨予報・警報級ならキャンセル規定を確認し旅行自体を再判断
